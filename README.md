# C--MotMystere

Principe du jeu "Le mot mystère"

1. Le joueur 1 saisit un mot au clavier ;
2. L'ordinateur mélange les lettres du mot ; 
3. Le joueur 2 essaie de deviner le mot d'origine à partir des lettres mélangées (il a droit à 5 essais) ;
4. Les joueurs peuvent rejouer à la fin d'une partie.
